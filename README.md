# Outreach Assistant

Мини‑приложение для B2B рассылок с готовыми шаблонами:
- база клиентов,
- библиотека шаблонов,
- кейсы с GitHub,
- массовая персонализация,
- экспорт кампаний в CSV/JSON.

## Как использовать

1. Заполните профиль оффера.
2. Добавьте кейсы вручную или импортируйте репозитории GitHub по username.
3. Добавьте клиентов.
4. Создайте шаблон с переменными:
   - `{{client_name}}`
   - `{{company}}`
   - `{{niche}}`
   - `{{offer}}`
   - `{{examples}}`
   - `{{owner_name}}`
   - `{{contact}}`
5. В разделе `Кампания` выберите шаблон и соберите рассылку.
6. Скачайте результаты в `CSV`/`JSON` или открывайте email по одному через `mailto`.

## Демо-данные

Кнопка `Загрузить демо` добавляет:
- 2 кейса GitHub,
- 2 клиентов,
- 1 рабочий шаблон.

## Хранение

Все данные сохраняются локально в браузере (`localStorage`).
