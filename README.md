# Outreach Assistant

Мини‑приложение для B2B рассылок с готовыми шаблонами:
- база клиентов,
- библиотека шаблонов,
- кейсы с GitHub,
- массовая персонализация,
- экспорт кампаний в CSV/JSON,
- реальная отправка email через Resend API,
- трекинг статусов `черновик → отправлено → прочитано → ответ`.

## Как использовать

1. Заполните профиль оффера.
2. Добавьте кейсы вручную или импортируйте репозитории GitHub по username.
3. Добавьте клиентов.
4. Создайте шаблон с переменными:
   - `{{client_name}}`
   - `{{company}}`
   - `{{niche}}`
   - `{{offer}}`
   - `{{examples}}`
   - `{{owner_name}}`
   - `{{contact}}`
5. В разделе `Кампания` выберите шаблон и соберите рассылку.
6. Скачайте результаты в `CSV`/`JSON` или открывайте email по одному через `mailto`.
7. Для автоотправки заполните `Vercel ENV` и используйте кнопки отправки в кампании.

## Демо-данные

Кнопка `Загрузить демо` добавляет:
- 2 кейса GitHub,
- 2 клиентов,
- 1 рабочий шаблон.

## Хранение

Все данные сохраняются локально в браузере (`localStorage`).

## Настройка отправки email

На Vercel добавьте переменные окружения:

- `RESEND_API_KEY` — API ключ Resend.
- `OUTREACH_FROM_EMAIL` — адрес отправителя (должен быть подтвержден в Resend).

Пример:

```bash
vercel env add RESEND_API_KEY production
vercel env add OUTREACH_FROM_EMAIL production
vercel --prod
```
